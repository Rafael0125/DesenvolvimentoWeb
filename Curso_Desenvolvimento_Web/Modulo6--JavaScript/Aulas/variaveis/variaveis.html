Tipos de variaveis

String

Variaveis de texto exemplo:
"var='Rafael'"

Number

São variáveis numéricas que são divididas em duas categorias:

"Int: Números inteiros, sem ponto, exemplo:
'var inteiro = 10' "

"Float: Números fracionados, com ponto, exemplo: br 'var fracionado = 10.66' "

Boolean

"Variáveis que permitem o armazenamento de estados, TRUE e FALSE, as variáveis do tipo boolean servem como parametro de testes condicionais."
